## Relationships

### 1. Inheritance (IS-A Relationship)
- Candidate **IS-A** Person
- Voter **IS-A** Person

### 2. Composition (HAS-A Relationship)
- VotingService **HAS-A** ArrayList of Candidates
- VotingService **HAS-A** HashMap of Voters
- Main **HAS-A** VotingService
- Main **HAS-A** Admin

### 3. Association
- Main **uses** VotingService
- Main **uses** Admin
- VotingService **manages** Candidates
- VotingService **manages** Voters

## OOP Concepts Mapping

### Encapsulation
```
Person class:
в”ңв”Җв”Җ private String id       вҶҗ Hidden
в”ңв”Җв”Җ private String name     вҶҗ Hidden
в”ңв”Җв”Җ private int age         вҶҗ Hidden
в””в”Җв”Җ public getters          вҶҗ Controlled access
```

### Inheritance
```
Person (Parent)
в”ңв”Җв”Җ Candidate (Child)
в””в”Җв”Җ Voter (Child)
```

### Abstraction
```
abstract class Person {
    abstract void displayInfo();  вҶҗ No implementation
}

class Candidate extends Person {
    void displayInfo() { ... }    вҶҗ Must implement
}
```

### Polymorphism
```
Person p1 = new Candidate(...);
Person p2 = new Voter(...);

p1.displayInfo();  вҶҗ Calls Candidate's version
p2.displayInfo();  вҶҗ Calls Voter's version
```
